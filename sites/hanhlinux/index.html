Hanh Linux

What exactly is this?

Hanh Linux is a Linux distro - based on Artix x64 and uses LFS/BLFS guide, focused on simplicity and performance.



Features

  • Simplicity: Hanh Linux tries to be minimal without removing a lot of base or network-related packages.
  • Designed for low-powered devices: Based on Artix Linux, it's much faster than most other popular linux distribtions. Moreover, Hanh Linux uses x64 gcc-libs and glibc instead of multilib, and openrc instead of systemd to make it usable on lower-powered devices.
  • systemd-free: systemd comes as a modern init daemon for Unix-like, but it does not follow UNIX philosophy, Hanh Linux uses openrc, a init with more powerful features.
  • Customization: Hanh Linux only give you base packages, a text editor and simple tools for network management, Users can decide what they want to install to their system.
