Easy to code

KodeLang and Pascal are the programming languages used to power Kode. Both of them are not difficult to experience and learn.

swap_vertical_circle

Save time, money,...

From one base code, Kode can be built and run on many platforms. This saves money, time and manpower.

Beutiful UI/UX

The interface and user experience of Kode-making applications are friendly and modern, in line with the design trend.

widgets

Diverse package

Packages are used for many purposes. In addition the community also contributes a lot of packages.

access_time

Fast

The software made by Kode is extremely fast operating speed on many platforms. This gives the best user experience.

Friendly tools

Kode provides friendly software development tools. There are both CLI and GUI.

Salient features

Command Line Tools

Interact with CLI - fast and convenient

Multiple IDE

You can use multiple IDEs to write Kode source code. The community is currently developing plugins for the IDE

JS and Kode

Kode is not only compatible with JS, but also compatible with most platforms for JS. If you are from JS then you will enjoy this.

Other language

Kode can combine many other languages. This increases the power and diversity.

Convert C to KodeLang

Can switch between the C language library to KodeLang. This increases the Kode power up.  

Kode Getter

Help Kode interact with the Server, download and install packages, display package information and lists from channels.

Database with Kode

Databases are used to store data, information, ... With Kode you can choose many different types of databases for your application.

Kode Live

Quick interaction with Kode through Live. In Live, you just need to press different buttons on the keyboard to control Kode to do everything.
